# 码云 ATOM 扩展

[码云](https://git.oschina.net) 是最大的中文开发者社区-开源中国 开发的的 GIT 代码托管平台. 
ATOM 是程序员非常喜爱的代码编辑器之一,并且拥有良好的插件生态,故码云团队通过悬赏推出
了码云的 ATOM 扩展.

## 如何使用

在命令行或者终端中运行:

```shell
apm install gitosc
```

或者可以打开设置页面安装 gitosc

![Install](http://git.oschina.net/GitGroup/atom-gitosc/raw/master/images/install.png)
